# Screenshot Instructions for F-Droid

F-Droid requires actual screenshots of your app. You need to:

1. Run your app on an Android device or emulator
2. Take screenshots of key features (2-8 screenshots recommended)
3. Screenshots should be in PNG format
4. Recommended size: 1080x1920 (phone portrait) or similar
5. Place them in: `fastlane/metadata/android/en-US/images/phoneScreenshots/`
6. Name them sequentially: `01.png`, `02.png`, etc.

## Key screens to capture:
- Main/home screen
- Bible study content view
- Scripture reference view
- Search functionality
- Topics/categories view

## How to take screenshots:
1. Run: `flutter run --release`
2. Use device screenshot feature or Android Studio's screenshot tool
3. Resize if needed to standard dimensions
